## Setup(Windows)
'''bash
# install environment
$ py -3 -m venv venv
$ venv\Scripts\activate
'''

##Running(Windows)
'''bash
# serve at default https://localhost:5000
$ venv\Scripts\activate
(venv) $ py tst.py

#leave venv
(venv) $ deactivate
'''
